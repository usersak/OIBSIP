Database Management

In this project, I designed and developed a comprehensive online event management system that streamlined event registration, coordination, and data management. The system utilizes a robust database to efficiently store and retrieve event-related information. I employed MySQL as the relational database management system to ensure data integrity and optimal querying performance. The system allows users to create accounts, browse upcoming events, register for events of their choice, and receive confirmation emails. Admin users have access to a separate interface where they can manage event details, attendee lists, and generate reports. To enhance security, I implemented role-based authentication, ensuring that only authorized users can access specific features.
